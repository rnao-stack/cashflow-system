## ⚠️ 기준 잔고 설정 (필수)

아래 값은 **초기 잔고 기준값**으로 사용됩니다.

```python
DEFAULT_BASE_DATE = "2026-07-05"  # 초기 잔고 기준 날짜
DEFAULT_BASE_CNY = 15102.45       # 초기 CNY 잔고
DEFAULT_BASE_USD = 1110.21         # 초기 USD 잔고
```

### 변경 시 주의사항

- **기준일 및 초기 잔고는 수동으로 확인 후 직접 수정하고 저장해야 합니다.**
- **소수점에는 천 단위 쉼표(,)를 사용하면 오류가 발생하므로 입력하지 마세요.**

예시
- ✅ `15102.45`
- ❌ `15,102.45`
